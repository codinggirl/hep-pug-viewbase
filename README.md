## 安装

- 运行 npm 命令行安装

```
npm i -S hep-pug-viewbase
```

或

```
npm install --save hep-pug-viewbase
```

- 运行 cnpm 命令行安装

```
cnpm i -S hep-pug-viewbase
```

或

```
cnpm install --save hep-pug-viewbase
```
